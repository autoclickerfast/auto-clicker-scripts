# Tap Tap Dig Automation Script

This script is specifically optimized for the idle mining clicker game *Tap Tap Dig*, designed to maximize your tap damage and fully automate heavy digging across the mining layers.

## 🌟 Features
* **Isometric Grid Multi-Target Tapping:** Deployed 25 precisely arranged target points distributed in a dense diamond-grid layout to fully cover the active digging zone for ultimate mining coverage.
* **Efficiency Optimization:** Finely tuned click delays to prevent the app from lagging while achieving the absolute technical speed limit of the clicker.
* **Auto Gold & Fossil Farming:** Strategic placement of targets ensures high-frequency mining damage while instantly collecting dropped coins, diamonds, and popping earthworms without finger strain.

## 🚀 How to Use
1. **Download the Script:** Download the [AutoClickerFast_TapTapBig.json](./AutoClickerFast_TapTapBig.json) file from this directory to your phone.
2. **Import Configuration:** Open your **AutoClickerFast** app, navigate to Configuration Management, and select **Import** to load the `.json` file.
3. **Launch the Game:** Open the *Tap Tap Dig* game, ensure your screen orientation matches, and press **Play** to start farming!
